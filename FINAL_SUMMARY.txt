🎬 DEMO FLOW
═══════════════════════════════════════════════════════════════════════════

$ python demo_load.py

You'll see:

1. Sequential Requests (3)
   └─ Each worker handles one request
   └─ WebUI shows distribution

2. Concurrent Spike (15)
   └─ All workers get requests simultaneously
   └─ Connections spike to 5+ each
   └─ Queue might fill temporarily

3. Steady Load (20s)
   └─ 3 requests/second sustained
   └─ Even distribution across workers
   └─ Stable latency trends

4. Variable Load (30s)
   └─ Quiet periods + spike periods
   └─ Queue fills during spikes
   └─ Clears during quiet times
